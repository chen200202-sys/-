# AI Agent Project - Information Systems 🤖
חן אבסדיד ועמיחי מזלתרים
פרויקט זה מציג סוכן בינה מלאכותית חכם.
האפליקציה מאפשרת שימוש בשני מצבים עיקריים:
1. **Chat Mode:** צ'אט רגיל לשיחות טקסט.
2. **Agentic Mode:** סוכן חכם המסוגל לבצע חיפוש ברשת (Web Search) בזמן אמת כדי לענות על שאלות אקטואליות.

* **Python:** שפת הפיתוח המרכזית.
* **Streamlit:** ספרייה לבניית ממשק משתמש אינטראקטיבי.
* **Requests:** לביצוע קריאות API מול שרת המכללה.
* **IAC Student API:** התשתית שסיפקה המכללה לגישה למודל.

## How to Run
כדי להפעיל את האפליקציה אצלכם במחשב, עקבו אחר הצעדים הבאים:

1. התקינו את הספריות הנדרשות:
   bash
   pip install requests streamlit

   הריצו את האפליקציה:
   streamlit run app.py
