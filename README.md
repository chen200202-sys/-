ברוכים הבאים לפרויקט סוכני ה-AI של עמיחי מזלתרים וחן אבסדיד 
הפרויקט מציג אבולוציה של מערכות בינה מלאכותית: החל מצ'אט בסיסי (Base) ועד לסוכן חכם (Agent) המסוגל לבצע חיפוש נתונים בזמן אמת ולנהל הקשר שיחה.

Base Chat (base.py): הקוד הבסיסי ביותר המדגים תקשורת פשוטה מול ה-API.

Stateless Agent (סוכן טיפש):

stupid_AI_pc: עובד בטרמינל, ללא זיכרון בין שאלות.

stupid_AI_ST: ממשק ויזואלי ב-Streamlit, ללא זיכרון.

Smart Agent (סוכן חכם):

smart_AI_pc: עובד בטרמינל, כולל יכולת חיפוש (Web Search) וניתוח נתונים.

SMART_AI_ST: ספינת הדגל! ממשק Streamlit מעוצב עם אייקונים, חיפוש ברשת וניהול היסטוריה.

Utilities:

TOKEN: קוד ייעודי למעקב וניטור ניצול הטוקנים היומי .

לפני שמריצים, ודאו שהתקנתם את הספריות הבאות:

Bash

pip install requests streamlit

איך זה עובד?
1. הרצה בטרמינל (PyCharm)
כדי להריץ את אחד מסוכני הטרמינל, פשוט הריצו את הקובץ ב-PyCharm או הקלידו בטרמינל:

Bash

python terminal_smart.py

2. הרצה בממשק ויזואלי (Streamlit)
כדי לפתוח את הממשק המעוצב בדפדפן, השתמשו בפקודה הבאה בטרמינל:

streamlit run streamlit_smart.py

תכונות מרכזיות שהוטמעו
Agentic Behavior: שימוש בכלי חיפוש חיצוניים לאימות מידע בזמן אמת.

Prompt Engineering: הנחיות (Instructions) ממוקדות לקבלת תשובות נקיות ללא לינקים ומקורות מציקים.

Context Management: שמירה על היסטוריית שיחה בתוך ה-Input כדי לאפשר שאלות המשך.

UI/UX: עיצוב מותאם אישית ב-Streamlit הכולל אייקונים (Avatars) ותצוגת סטטוס טוקנים צבעונית.
